Questão 14: Comparação entre localStorage, sessionStorage e IndexedDB

Diferenças Principais

Característica localStorage sessionStorage IndexedDB
Volume de Dados ~5MB ~5MB ~50MB ou mais
Formato de Armazenamento Chave/Valor (strings) Chave/Valor (strings) Banco de dados NoSQL (objetos estruturados)
Persistência Persistente (até ser explicitamente apagado) Por sessão (dados são apagados ao fechar a aba/navegador) Persistente
Casos de Uso Dados simples (tokens, preferências) Dados temporários (dados de formulário durante a navegação) Dados complexos e volumosos (listas, arquivos)
